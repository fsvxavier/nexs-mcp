# NEXS MCP Examples

This directory contains example scripts and configurations for using NEXS MCP Server.

## Directory Structure

- **basic/** - Basic usage examples (create, list, update elements)
- **integration/** - Integration examples with Claude Desktop
- **workflows/** - Complete workflow examples

## Getting Started

1. Make sure NEXS MCP Server is installed and running
2. Review the examples in each directory
3. Modify the examples to fit your use case
4. Run the scripts to see NEXS MCP in action
