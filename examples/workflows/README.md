# Workflow Examples

Complete workflow examples demonstrating end-to-end usage of NEXS MCP.

## Available Workflows

- `complete_workflow.sh` - Complete workflow from setup to execution

## Workflow Overview

A typical NEXS MCP workflow includes:

1. **Setup**: Create personas, skills, and templates
2. **Organization**: Tag and categorize elements
3. **Execution**: Use ensembles for complex tasks
4. **Management**: Backup, sync with GitHub
5. **Analysis**: Review usage statistics and performance

## Running Workflows

```bash
chmod +x complete_workflow.sh
./complete_workflow.sh
```
