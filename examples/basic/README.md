# Basic Examples

Basic examples demonstrating core NEXS MCP operations.

## Available Scripts

- `create_persona.sh` - Create a sample persona
- `create_skill.sh` - Create a sample skill
- `list_elements.sh` - List all elements
- `create_ensemble.sh` - Create a sample ensemble

## Usage

Make sure NEXS MCP Server is running, then execute any script:

```bash
./create_persona.sh
./list_elements.sh
```

## Notes

These examples use the MCP protocol over stdio. Make sure the server is configured to accept connections.
